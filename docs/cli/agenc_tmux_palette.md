## agenc tmux palette

Open the AgenC command palette (runs inside a tmux display-popup)

### Synopsis

Presents an fzf-based command picker inside a tmux display-popup.
On selection, the chosen command runs in a new tmux window. On cancel
(Ctrl-C or Esc), the popup closes with no action.

This command is designed to be invoked by the palette keybinding
(prefix + a, Space) and requires --parent-pane.

```
agenc tmux palette [flags]
```

### Options

```
  -h, --help                 help for palette
      --parent-pane string   Parent pane ID (required, passed from keybinding via #{pane_id})
```

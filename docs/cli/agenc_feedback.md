## agenc feedback

Launch a feedback mission with Adjutant

### Synopsis

Launches a new Adjutant mission for sending feedback about AgenC.
This is a shorthand for:
  agenc mission new --adjutant --prompt "I'd like to send feedback about AgenC"

```
agenc feedback [flags]
```

### Options

```
  -h, --help   help for feedback
```

## agenc config paletteCommand add

Add a custom palette command

### Synopsis

Add a custom palette command to config.yml.

The name is used as the config key and must start with a letter, containing
only letters, numbers, hyphens, and underscores (max 64 characters).
Built-in names cannot be used ‚Äî use 'update' to override builtins.

Commands that reference $AGENC_CALLING_MISSION_UUID are "mission-scoped" ‚Äî
they only appear in the palette when the focused pane is running a mission.

The keybinding value is passed through to tmux's bind-key command. By default,
keybindings live in the AgenC key table (prefix + a, <key>). To make a global
binding that works anywhere without the AgenC leader, prefix with "-n":

  --keybinding="f"       ‚Üí prefix + a, f  (AgenC table)
  --keybinding="-n C-s"  ‚Üí Ctrl-s globally (root table, no prefix needed)

    --command="agenc tmux window new -- agenc mission new mieubrisse/dotfiles" \
    --keybinding="f"

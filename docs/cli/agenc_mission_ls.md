## agenc mission ls

List active missions

```
agenc mission ls [flags]
```

### Options

```
  -a, --all           include archived missions
      --cron string   filter to missions from a specific cron job
      --git-status    show uncommitted changes in mission directories
  -h, --help          help for ls
```
